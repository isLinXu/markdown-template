## 训练计划

| 星期 | 训练项目       | 组数 | 每组次数 | 备注          |
| ---- | -------------- | ---- | -------- | ------------- |
| 一   | 有氧运动（跑步） | -    | 30分钟   | 适量热身和拉伸 |
| 二   | 上肢训练       | 4    | 8-12     | 包括卧推、引体向上等 |
| 三   | 有氧运动（游泳） | -    | 45分钟   | 适量热身和拉伸 |
| 四   | 下肢训练       | 4    | 8-12     | 包括深蹲、硬拉等 |
| 五   | 有氧运动（骑行） | -    | 1小时    | 适量热身和拉伸 |
| 六   | 核心训练       | 4    | 10-15    | 包括仰卧起坐、俯卧撑等 |
| 日   | 休息           | -    | -        | 适当放松和休息 |
